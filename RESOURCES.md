# Vulkan Rendering Engine Resources

## Knowledge

- [LunarG: Getting Started with the Windows Vulkan SDK](https://vulkan.lunarg.com/doc/view/latest/windows/getting_started.html)
  Use for checking SDK installation, environment variables, `vulkaninfo`, and Windows SDK layout.
- [Khronos Vulkan Tutorial: Development Environment](https://docs.vulkan.org/tutorial/latest/02_Development_environment.html)
  Use for the modern Khronos tutorial path and the basic tool/library setup.
- [Khronos Vulkan Tutorial repository](https://github.com/KhronosGroup/Vulkan-Tutorial)
  Use when you want to compare the newer Vulkan 1.4 teaching style and code organization.
- [Vulkan Guide](https://docs.vulkan.org/guide/latest/)
  Use for concept explanations once tutorial code starts feeling too recipe-like.
- [GLFW Vulkan Guide](https://www.glfw.org/docs/latest/vulkan_guide.html)
  Use for window surface creation, required instance extensions, and GLFW-specific Vulkan behavior.
- [RenderDoc](https://renderdoc.org/)
  Use once you need frame capture, pipeline inspection, texture inspection, and draw-call debugging.
- [Original Vulkan Tutorial](https://vulkan-tutorial.com/)
  Use as a beginner-friendly reference, but cross-check with Khronos docs when APIs or best practices differ.

## Wisdom (Communities)

- [Khronos Vulkan Forum](https://community.khronos.org/c/vulkan/11)
  Use for API-level questions and validation-layer mysteries.
- [r/vulkan](https://www.reddit.com/r/vulkan/)
  Use for practical learning questions, examples, and feedback from other Vulkan learners.
